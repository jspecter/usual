# About cnmjs
 a simple cli help us to auto generate some usual config files. such as editor,eslint,git,prettier,ts and so on. it's out of the box but not proper to old hand.
---

## **cnmjs docs**

---

### install

```
yarn global add cnmjs
```

or

```
npm install cnmjs -g
```

### usage

```
cnmjs
```
